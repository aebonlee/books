# 데이터베이스 실습 워크북

## 실습 1: SQL 기초

### 테이블 생성

```sql
CREATE TABLE students (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  email VARCHAR(255) UNIQUE,
  enrolled_at TIMESTAMP DEFAULT NOW()
);
```

### 데이터 조회

```sql
SELECT name, email
FROM students
WHERE enrolled_at > '2025-01-01'
ORDER BY name ASC;
```
