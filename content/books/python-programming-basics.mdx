# 파이썬 프로그래밍 기초

## 1장: 파이썬 소개

파이썬은 1991년 귀도 반 로섬이 개발한 프로그래밍 언어입니다. 읽기 쉽고 간결한 문법으로 전 세계에서 가장 인기 있는 프로그래밍 언어 중 하나입니다.

### 1.1 파이썬의 특징

- **간결한 문법**: 읽기 쉬운 코드 작성 가능
- **다양한 라이브러리**: 풍부한 표준 라이브러리와 서드파티 패키지
- **플랫폼 독립적**: Windows, macOS, Linux에서 동일하게 실행

### 1.2 개발 환경 설정

```python
# 파이썬 버전 확인
import sys
print(f"Python {sys.version}")

# Hello World
print("안녕하세요, 파이썬!")
```

## 2장: 변수와 데이터 타입

파이썬에서 변수는 데이터를 저장하는 이름표입니다.

```python
# 변수 선언
name = "김철수"
age = 25
height = 175.5
is_student = True

# 데이터 타입 확인
print(type(name))    # <class 'str'>
print(type(age))     # <class 'int'>
```

> **참고**: 파이썬은 동적 타입 언어로, 변수의 타입을 명시적으로 선언할 필요가 없습니다.
