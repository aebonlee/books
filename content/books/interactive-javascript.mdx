# 인터랙티브 자바스크립트 교과서

## 1장: 자바스크립트란?

자바스크립트는 웹의 프로그래밍 언어입니다. 브라우저에서 실행되며, Node.js를 통해 서버에서도 사용됩니다.

## 2장: 변수와 상수

```javascript
// let: 변경 가능한 변수
let greeting = "안녕하세요";
greeting = "Hello";

// const: 변경 불가능한 상수
const PI = 3.14159;

// 템플릿 리터럴
const name = "김철수";
console.log(`${name}님, 환영합니다!`);
```

## 3장: 함수

```javascript
// 화살표 함수
const add = (a, b) => a + b;

// 고차 함수
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(n => n * 2);
console.log(doubled); // [2, 4, 6, 8, 10]
```

> 위의 코드를 직접 수정하고 실행해보세요!
